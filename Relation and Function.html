Relation

Any set of ordered pairs.
The set of all x-values is called domain of the relation. While the Set of all y-values is called range of the relation.

examples of sets of ordered pairs:
{(3,6),(13,15),(13,14),(11,12)}
{(2,8),(3,7),(5,8)(0,11)}
{(8,9),(3,4),(1,4),(3,5)}
